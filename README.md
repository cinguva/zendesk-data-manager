## Development

Data Exporter is a Vue.js 2 app built on the Zendesk Apps Framework v2.0.

### Build Setup

``` bash
# Install dependencies
npm install

# Builds application
npm run build

# Run ZAT server, navigate to https://roseware.zendesk.com/agent/apps/data-manager?zat=true
cd app && zat server
```
